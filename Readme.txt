
Gerador de senha

O código cria uma interface gráfica simples para gerar senhas aleatórias e 
salva informações relacionadas a essas senhas em um arquivo de texto. 
Ele usa a biblioteca PySimpleGUI para criar a interface gráfica e a biblioteca 
random para gerar senhas aleatórias. Além disso, ele reproduz um som no início 
do programa.



Password generator

The code creates a simple graphical interface to generate random passwords and
saves information related to these passwords to a text file.
It uses the PySimpleGUI library to create the graphical interface and library
random to generate random passwords. Additionally, it plays a sound at the beginning
from the program.
